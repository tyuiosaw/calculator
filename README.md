# Travel Calc

## Описание
Travel Calc — одностраничное веб-приложение для выполнения базовых вычислений, необходимых в путешествиях: конвертация валют, преобразование единиц измерения и расчёт бюджета.

Приложение не использует сторонние библиотеки и работает полностью на нативных веб-технологиях.

---

## Функциональность

### 1. Конвертер валют
- Поддерживаемые валюты: RUB, USD, EUR, GBP, CNY, TRY
- Динамическая загрузка курсов (через внешний API в `script.js`)
- Ручной ввод суммы и выбор направлений конвертации
- Отображение результата в UI

### 2. Конвертер единиц

#### Расстояние
- miles → kilometers
- kilometers → miles

#### Вес
- pounds → kilograms
- kilograms → pounds

Пересчёт выполняется при событиях `input` и `change`.

### 3. Менеджер бюджета
- Добавление позиции: название, сумма, валюта
- Конвертация всех значений в базовую валюту (RUB)
- Отображение списка через `<table>`
- Подсчёт общей суммы
- Удаление элементов

---

## Архитектура

Приложение построено по принципу разделения:

- **index.html** — структура DOM
- **style.css** — визуальное оформление (Grid + Flexbox)
- **script.js** — бизнес-логика и работа с DOM

Без использования:
- сборщиков (Webpack, Vite и т.д.)
- фреймворков (React, Vue и др.)
- сторонних UI-библиотек
## Запуск

```bash
open index.html
```

Или открыть файл вручную в браузере.
